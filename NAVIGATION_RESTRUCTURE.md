## 🚀 사용자 플로우

### 플로우 #1: 모의 면접
```
1. 대시보드 → "🎤 모의 면접 시작하기" 클릭
2. /interview 페이지 이동
3. 과거 피드백 선택 또는 새 면접 시작
4. 질문 듣기 → 음성 답변 → 피드백 수신
5. 결과 저장 → /history에서 확인 가능
```

### 플로우 #2: 자기소개서 피드백
```
1. 대시보드 → "📝 자기소개서 피드백 받기" 클릭
2. /resume 페이지 이동
3. Step 1: 채용 공고 입력 및 분석
4. Step 2: 자기소개서 작성/붙여넣기
5. AI 피드백 수신 → /feedback/[id]로 이동
6. 결과 확인 → /history에서 재확인 가능
```
